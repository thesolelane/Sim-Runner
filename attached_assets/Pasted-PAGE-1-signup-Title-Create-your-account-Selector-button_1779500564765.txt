Summary — minimum sim path
/signup          → click "Contractor / Tradesperson" → Continue
/signup/trade-select → click any trade → "Continue as [trade]"
/signup/contractor step 1 → email + password + confirm → Continue
/signup/contractor step 2 → full name (required) + trade select (required) → Continue
/signup/contractor step 3 → (all optional, can submit blank) → "Create Account"
/wallet-setup    → wait for page load → click "I've saved my key — continue"
/feed            → ✅ done

The only required fields across all steps:

Email, Password, Confirm Password (step 1)
Full Name, Primary Trade (step 2)
Everything on step 3 is optional
