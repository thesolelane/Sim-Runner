# Cooperanth Sim Runner

Universal simulation engine for testing user journeys across any app.

## Quick Start

```bash
npm install
npx playwright install chromium
npm run test:contractor
```

## Usage

```bash
# Test specific user type
node run.js traydbook contractor
node run.js traydbook owner
node run.js traydbook agent

# Watch browser (headed mode)
node run.js traydbook contractor --headed

# Test all user types
npm run test:all
```

## Adding Your App

Edit `config.js`:

```javascript
export const APPS = {
  traydbook: { ... },
  yourapp: {
    url: 'https://yourapp.com',
    type: 'saas',
    users: {
      admin: { role: 'admin', flow: 'signup → dashboard → settings' },
      user: { role: 'user', flow: 'signup → onboarding → useFeature' }
    }
  }
};
```

Then add corresponding methods to `engine.js`.

## CI/CD Integration

### GitHub Actions
```yaml
- name: Run Cooperanth Sim
  run: |
    npm install
    npx playwright install chromium
    npm run test:all
```

### Coolify Scheduled Task
```bash
0 2 * * * cd /path/to/sim-runner && node run.js traydbook contractor
```
